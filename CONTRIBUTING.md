# Contributing to faang-wolfram-solver

Thank you for your interest in contributing!

## Development Setup

1. Fork and clone the repository
2. Install Rust and Node.js
3. Run `make setup`
4. Start development with `make dev`

## Code Style

- **Rust**: Follow rustfmt conventions
- **TypeScript**: Follow ESLint rules
- **Commits**: Use conventional commits (`feat:`, `fix:`, `docs:`, etc.)

## Pull Request Process

1. Create a feature branch from `develop`
2. Make your changes
3. Add tests if applicable
4. Ensure all tests pass (`make test`)
5. Update documentation if needed
6. Submit PR to `develop` branch

## Code Review

All submissions require review before merging. We use GitHub PRs for this process.
